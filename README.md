# Rosewal Almeida — Data Analyst Portfolio

Professional portfolio website built with vanilla HTML, CSS, and JavaScript. Dark theme inspired by [nitinredhu.github.io](https://nitinredhu.github.io). Zero dependencies.

## Structure
- **Hero** — Name, tagline, availability badge, 4 key metrics from real experience
- **Toolkit** — 6-card skills grid
- **Projects** — 6 projects with Problem → Action → Result → Impact format, real GitHub links + live Power BI dashboards
- **Experience** — 4-role career timeline with dates, companies, and quantified achievements
- **Contact** — Email, GitHub, LinkedIn, Carrd
- **Resume Modal** — Full CV in popup overlay

## Deploy to GitHub Pages
1. Go to `github.com/new` → name: `portfolio` → Public → Create
2. Upload `index.html`, `style.css`, `script.js`, `README.md`
3. Settings → Pages → Deploy from branch → `main` / `(root)` → Save
4. Live at: `roswel0710.github.io/portfolio`

## Tech Stack
- HTML5 + SEO meta tags + Open Graph
- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript
- Google Fonts: Inter + Sora
