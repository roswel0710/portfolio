# Roswel Almeida — Data Analytics Portfolio

A professional dark-themed portfolio website inspired by [nitinredhu.github.io](https://nitinredhu.github.io). Built with vanilla HTML, CSS, and JavaScript — zero dependencies.

## 🚀 Deploy to GitHub Pages

### Step 1: Create the Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it `portfolio` (or `roswel0710.github.io` for a root-level site)
3. Set to **Public**
4. Click **Create repository**

### Step 2: Upload Files
```
├── index.html       ← Main page
├── style.css        ← Dark theme styles
├── script.js        ← Interactions & animations
└── README.md        ← This file
```

**Option A — GitHub UI:**
Click "Add file" → "Upload files" → drag all files → commit.

**Option B — Git CLI:**
```bash
git init
git add .
git commit -m "Initial portfolio"
git branch -M main
git remote add origin https://github.com/roswel0710/portfolio.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Repository **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Click **Save**

### Step 4: Go Live
Your site will be at: `https://roswel0710.github.io/portfolio/`

## 📁 Structure
- **Hero** — Name, tagline, availability badge, key metrics
- **Toolkit** — 6-card skills grid
- **Projects** — 4 cards with Problem → Action → Result → Impact format
- **Live Dashboard** — KPIs, bar chart, trend line (static demo)
- **Contact** — Email, GitHub, LinkedIn, Carrd links
- **Resume Modal** — Full CV in overlay popup

- CSS3 (custom properties, grid, flexbox, animations)
- Vanilla JavaScript (no frameworks)
- Google Fonts: Inter + Sora
